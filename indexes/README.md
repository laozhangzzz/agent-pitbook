# Indexes

`indexes/` contains rebuildable human and agent navigation aids.

Indexes are not the source of truth. They can be regenerated from `pits/`, `claims/`, and `sources/`.

Recommended indexes:

- `index.md`: global pit map
- `tags.md`: tag map
- `tools.md`: affected tool map
- future: `claims.md`, `sources.md`, `stale.md`
